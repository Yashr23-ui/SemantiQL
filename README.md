SemantiQL
SemantiQL is an academic SQL query compiler front‑end designed to demonstrate compiler design principles applied to SQL query processing. The system analyzes SQL queries and processes them through multiple stages such as lexical analysis, parsing, semantic validation, intermediate code generation, optimization, and logical execution planning.

This project is developed as part of a Compiler Design Project Based Learning (PBL) to help understand how database systems internally process SQL queries.

Project Objectives
Demonstrate compiler pipeline concepts using SQL queries

Implement lexical analysis for SQL tokens

Build a recursive descent parser

Generate an Abstract Syntax Tree (AST)

Perform semantic validation using a symbol table

Generate Three Address Code (TAC)

Apply rule‑based query optimization

Produce a logical execution plan
